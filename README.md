# Birko.Data.Tests

Unit tests for the core Birko.Data project.

## Test Coverage

- **AsyncStoreTests** - CRUD operations, Save, Init, Destroy, CancellationToken support

## Test Framework

- xUnit 2.9.3
- FluentAssertions 7.0.0
- .NET 10.0

## Running Tests

```bash
dotnet test Birko.Data.Tests
```

## Dependencies

- Birko.Data (shared project via .projitems)
